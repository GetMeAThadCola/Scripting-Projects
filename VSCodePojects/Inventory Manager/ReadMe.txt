Inventory Management System with Tkinter and SQLite
This project is an inventory management system designed to maintain records of various items using a Graphical User Interface (GUI) created with Tkinter in Python. The system stores its data in a SQLite database, allowing users to add, delete, search, and manage inventory items efficiently.

Objective
The primary objective of this project was to build a user-friendly application that enables users to maintain an inventory record and manage their items effortlessly. The project utilizes the Tkinter library to create an intuitive graphical interface and SQLite for database operations.

Features
Add Items: Enables users to input details and add items to the inventory database.
Delete Items: Provides functionality to delete selected items from the inventory list.
Search Inventory: Allows users to search and filter inventory items based on specified criteria.
Skills and Technologies Utilized
Tkinter: Utilized for developing the GUI components and user interaction.
SQLite: Employed for creating and managing the database of inventory items.
Python: The core programming language used to create the application logic and handle data operations.
Future Improvements
Potential enhancements could involve implementing additional features like item categorization, user authentication, and data visualization through graphs or statistics
