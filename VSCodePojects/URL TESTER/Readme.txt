
URL Reachability and Traceroute Checker

This Python script provides a simple GUI application for checking the reachability of a URL and performing a traceroute to the domain or IP address associated with the URL.

Features:

Enter a URL to check its reachability using the requests library.
Perform a traceroute to the domain or IP address extracted from the URL using the scapy library.
Display the results of the reachability check and traceroute in a user-friendly graphical interface.
This tool helps you quickly verify the accessibility of a URL and trace the path taken by network packets to reach the destination.

Requirements:

Python 3.x
Libraries: requests, scapy, tkinter
Usage:

1. Run the script to open the GUI application.
2. Enter a URL in the provided text box.
3. Click the "Check Reachability" button.
4. The application will display the reachability status and the intermediate IP addresses in the traceroute.
5. Please make sure to install the required libraries before using the script.
